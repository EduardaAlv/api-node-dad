Português
-----Trabalho de Desenvolvimento de Aplicações Distribuídas-----

----Exemplo de projeto Node.js, foi realizada a criação de uma API do tipo GET----
----Iniciando o servidor node----
1) Instalar todas as dependências necessárias para o projeto.
2) Executar no terminal o comando: node server.js
3) Para acessar a lista com todos os produtos: http://{porta}/api/produtos
4) Para acessar um produto específico: http://{porta}/api/produtos/{id}

English
-----Distributed Application Development Work-----

----Example of Node.js project, creation of an API of type GET----
----Starting the server node----
1) Install all required dependencies for the project.
2) Run the command in the terminal: node server.js
3) To access the list of all products: http://{port}/api/products
4) To access a specific product: http://{port}/api/products/{id}
