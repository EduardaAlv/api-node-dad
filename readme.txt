----Exemplo no Node.js, foi realizada a criação de uma API do tipo Get----
----Iniciando o servidor node----
1) Executar no terminal o comando: node server.js
2) Para acessar a lista com todos os produtos: http://127.0.0.1:3000/api/produtos
3) Para acessar um produto específico: http://127.0.0.1:3000/api/produtos/{id}
